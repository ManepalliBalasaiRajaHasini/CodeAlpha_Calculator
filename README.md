# CodeAlpha_Calculator
# Calculator Web Application

## 📌 Project Overview

This project is a modern Calculator Web Application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations and features a responsive Glassmorphism user interface.

## 🚀 Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear Screen (C)
* Delete Last Character (⌫)
* Real-Time Calculation
* Responsive Design
* Glassmorphism UI
* Hover Animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

CodeAlpha_Calculator/
│
├── index.html
├── style.css
├── script.js
└── README.md

## 🎯 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.

## 📸 Output

The application provides a clean and modern calculator interface with a glassmorphism design and supports all basic arithmetic operations.
